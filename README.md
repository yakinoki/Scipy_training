## GitHub Actions

このリポジトリは GitHub Actions の練習用に構成されており、Python スクリプトに対して構文チェック（flake8）を自動で行います。

`.github/workflows/python-app.yml` にて定義された CI パイプラインは、`push` および `pull_request` のたびに実行されます。
